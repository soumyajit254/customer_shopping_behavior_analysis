# customer_shopping_behavior_analysis
End-to-end data analytics project using Python, PostgreSQL, and Power BI, covering data cleaning, EDA, SQL analysis, and interactive dashboard creation.
Workflow
Data Loading
•	Imported dataset using Pandas 
•	Loaded cleaned data into PostgreSQL using SQLAlchemy
Exploratory Data Analysis (EDA)
•	Checked missing values and duplicates 
•	Analyzed distributions of key variables 
•	Identified trends and patterns
Data Cleaning
•	Handled missing values 
•	Removed duplicates 
•	Standardized column formats 
•	Validated data consistency
SQL Analysis (PostgreSQL)
Performed advanced queries such as:
•	Revenue analysis by category 
•	Customer segmentation 
•	Purchase trends
Power BI Dashboard
Developed an interactive dashboard including:
•	Sales by Category
•	Revenue by Category
•	Revenue by Age Group
•	Percentage of Customer by Subscription Status
•	Sales by Age Group
